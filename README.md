# mdb-todo-app

  Very simple todo app in React which I created as a recruitment task.

## Installation

```bash
git clone git@github.com:damianziobro/mdb-todo-app.git
```

```bash
cd mdb-todo-app
npm install
```

## Development

```bash
npm run start
```

```bash
http://localhost:3000
```

## Production

```bash
npm run build
```
